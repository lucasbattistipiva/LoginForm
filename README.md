# 🔐 Formulário de Login

Um projeto simples de uma página de formulário desenvolvido com **HTML5** e **CSS3**, focado em praticar a criação de formulários responsivos, validação nativa dos campos e organização da estrutura da página.

---

## 📸 Demonstração

**Acesse o projeto:**  
🔗 **Link do site:** https://loginformulari.netlify.app/

---

## 🚀 Funcionalidades

- ✅ Campo para E-mail
- ✅ Campo para Senha
- ✅ Campo para Username
- ✅ Campo para Nome
- ✅ Campo para Sobrenome
- ✅ Campo para CPF
- ✅ Checkbox para aceitar os Termos de Condição
- ✅ Validação nativa utilizando HTML5
- ✅ Campos obrigatórios (`required`)
- ✅ Validação mínima de caracteres (`minlength`)
- ✅ Layout organizado e responsivo

## 📋 Validações

O formulário utiliza as validações nativas do HTML:

| Campo | Validação |
|--------|-----------|
| E-mail | Formato de e-mail válido |
| Senha | Mínimo de 8 caracteres |
| Username | Mínimo de 3 caracteres |
| Nome | Mínimo de 3 caracteres |
| Sobrenome | Mínimo de 4 caracteres |
| CPF | Mínimo de 11 caracteres |
| Termos | Obrigatório aceitar |

## 🎯 Objetivo

Este projeto foi criado com o objetivo de praticar:

- Estruturação de formulários em HTML;
- Utilização de diferentes tipos de `<input>`;
- Validações nativas do navegador;
- Organização do código;
- Desenvolvimento de interfaces simples para autenticação.
